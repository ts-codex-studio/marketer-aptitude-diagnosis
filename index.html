Precision Edition for Students

マーケター素質診断

将来マーケターを考えている学生・新卒向けに、知識やスキルではなく、 「どんな見方を自然にしてしまうか」を見る診断です。今回は精度を優先して、 Quick Pulse を外し、その分の設問数を Core Scale と Mini Cases に再配分しています。

5つのコア素質 Core Scale を増問 ケース選択で反応のクセを見る 異質接触は補助指標 型は断定でなく「寄り」で返す

今回の改善ポイント

向いているかを乱暴に決めるのでなく、マーケの仕事を面白がりやすい認知特性があるかを、少し丁寧に見にいく版です。

1. コア素質中心

観察・因果・他者視点・表現最適化・反応学習の5軸を主判定にしています。

2. 設問を後段へ再配分

情報量の少ない Yes / No を削り、その分を主判定とケース反応に振り向けています。

3. 見せ方バイアスを減らす

逆転項目とケース設問で、きれいに答えるだけでは高得点になりにくくしています。

Part 2

Core Scale 20問

主判定パートです。1 = 全く当てはまらない / 5 = 完全に当てはまる で答えてください。逆転項目も入っています。

Part 3

Mini Cases 10問

もしその場面にいたら、最初に一番しそうな反応を選んでください。自己評価より、反応スタイルのクセを見ます。

Supplement

異質接触 3問

これは補助指標です。総合判定には入れず、今の環境がどれだけ視点を広げやすいかを見る参考に使います。
